Ping Tool

Description
This tool sends an ICMP ping to an IP address to check if the address is reachable.

Installation
This tool requires the scapy package. Install it by running:
pip install scapy

Or, install it by navigating to the tool's directory and running:
pip install .

Usage
To use the ping tool, specify the IP address as follows:
python pinger.py 192.168.1.1
This command will check if the specified IP address is reachable.
